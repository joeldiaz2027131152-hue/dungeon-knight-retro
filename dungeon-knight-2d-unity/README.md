# Dungeon Knight Unity Port

Primer vertical slice del port a Unity 2D usando el mundo 1-1 como base.

## Como abrirlo

1. Abre Unity Hub.
2. Usa **Add project from disk**.
3. Selecciona la carpeta `dungeon-knight-2d-unity`.
4. Abre el proyecto con Unity 2022 LTS o superior.
5. Crea o abre una escena vacia y pulsa Play.

El script `WorldOneOneBootstrap` construye una escena de prueba automaticamente al iniciar Play.

## Controles

- `A/D` o flechas: moverse
- `Space`: saltar
- `J`: atacar
- `K`: escudo
- `L`: rodar
- `E`: interactuar con cofres, hogueras y cartas

## Que incluye este primer paso

- Player 2D con movimiento, salto, ataque, escudo, rodada y stamina.
- Vida y stamina basicas.
- Enemigo simple que persigue al verte.
- Cofre con monedas.
- Hoguera que cura vida y stamina.
- Carta/lore inicial del mundo 1-1.
- Plataformas placeholder para empezar a sentir el ritmo del nivel.

La idea es iterar aqui hasta que se sienta bien y luego reemplazar placeholders por sprites/animaciones mejores.
