# Dungeon Knight HTML

Version base original del juego, hecha con HTML5 Canvas, CSS y JavaScript.

## Como abrir

Abre `index.html` en el navegador desde esta carpeta.

## Estructura

- `index.html`: interfaz y canvas del juego.
- `style.css`: estilos, HUD y efectos visuales.
- `js/`: motor, personaje, enemigos, boss, audio y particulas.
