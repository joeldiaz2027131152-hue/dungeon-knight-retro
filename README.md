# Dungeon Knight

Este repositorio tiene dos versiones separadas del juego para evitar mezclar archivos:

## Carpetas

- `dungeon-knight-retro-html/`: juego base original en HTML5 Canvas y JavaScript.
- `dungeon-knight-2d-unity/`: port del juego a Unity 2D.

## Como abrir cada version

### Version HTML

Abre `dungeon-knight-retro-html/index.html` en el navegador, o sirve esa carpeta con un servidor local.

### Version Unity

Abre Unity Hub, usa **Add project from disk** y selecciona la carpeta `dungeon-knight-2d-unity/`.

## Nota

Los cambios nuevos del port 2D se estan haciendo dentro de `dungeon-knight-2d-unity/`. La version HTML queda separada como referencia/base original.
