# Dungeon Knight

Este repositorio tiene dos versiones separadas del juego para evitar mezclar archivos:

## Carpetas

- `web-html/`: juego base original en HTML5 Canvas y JavaScript.
- `unity-port/`: port del juego a Unity 2D.

## Como abrir cada version

### Version HTML

Abre `web-html/index.html` en el navegador, o sirve esa carpeta con un servidor local.

### Version Unity

Abre Unity Hub, usa **Add project from disk** y selecciona la carpeta `unity-port/`.

## Nota

Los cambios nuevos del port se estan haciendo dentro de `unity-port/`. La version HTML queda separada como referencia/base original.
